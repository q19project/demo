# 如何写Demo

本项目旨在教育初学者，如何以最高的效率编写一个demo项目。

## 写在前面

我是一个写web页面demo项目的高手，人称“demo帝”，所以本项目旨在教育初学者们如何更好更快的写一个demo出来。

名词解释：

**demo**：源自唱片页，歌手录一小段音频小样发给唱片公司供人拣选。本项目中特指软件行业中一个可以供吃瓜群众评头论足的原型、示范等等。
